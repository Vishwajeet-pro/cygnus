Cygnus is a modern, 5-page responsive website for a fictional FinTech (financial technology) app. This project is built using only **HTML**, **CSS**, and **vanilla JavaScript**.

The design focuses on a sleek, **dark-mode** aesthetic, utilizing CSS variables, gradients, and "glassmorphism" for the navbar. It also features subtle reveal-on-scroll animations to create a professional and dynamic user experience.

---

## âœ¨ Features

* **5-Page Structure:**
    * `Home`: A high-impact landing page.
    * `Features`: A detailed breakdown of product features.
    * `Pricing`: A 3-column pricing table.
    * `About Us`: A page for the company's mission and team.
    * `Contact Us`: A functional contact form.
* **Modern Design:**
    * **Dark Mode First:** A sleek, tech-oriented color palette.
    * **Glassmorphism Navbar:** A sticky, semi-transparent blurred navigation bar.
    * **Fully Responsive:** Looks great on all devices, from mobile to desktop, using Flexbox, Grid, and media queries.
* **JavaScript Interactivity:**
    * **Responsive Hamburger Menu:** A clean, functional menu for mobile devices.
    * **Form Validation:** Client-side validation for the contact form.
    * **Reveal-on-Scroll:** Sections and cards fade and slide into view as the user scrolls, powered by the `IntersectionObserver` API.
